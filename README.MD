## 📖 About
**Tsukuri** is a minimalist, high-performance personal productivity suite designed for those who value consistency over complexity. By combining a visual habit tracker with a streamlined digital journal, Tsukuri helps you document your progress and cultivate a mindset of continuous improvement.

## ✨ Features
*   **Visual Habit Heatmap**: A clean, intuitive dashboard to visualize your consistency and streaks over the course of a year.
*   **Integrated Journaling**: A lightweight, distraction-free diary to capture daily reflections, thoughts, and moods.
*   **Local-First Architecture**: Your data stays with you. Everything is stored locally in your browser, ensuring maximum privacy and speed.
*   **Responsive Design**: Optimized for both desktop and mobile, so you can track your habits and write your journals on the go.

## 🛠 Tech Stack
*   **Frontend**: Pure HTML5, CSS3, and Vanilla JavaScript.
*   **Backend**: Flask.
*   **Storage**: Browser LocalStorage for persistent, offline-ready data management.
*   **UI/UX**: Minimalist aesthetic focusing on a dark-mode-first experience using 'DM Sans'.

## 🚀 Getting Started

### Prerequisites
You only need a modern web browser and a local server environment (Flask) to get started.

### Installation
1. Clone the repository:
```bash
   git clone [https://github.com/galacticpsychopath/tsukuri.git](https://github.com/galacticpsychopath/tsukuri.git) .
```
2. Navigate to the project directory:
```bash
   cd tsukuri
```
3. Run the application using your Flask environment or open the static files directly in your browser:
```bash
   python backend.py
```

## 💡 Philosophy
The name Tsukuri (造り) embodies the spirit of "making" or "building." We believe that personal growth is not about radical shifts, but the daily, intentional construction of habits that define who we are.
